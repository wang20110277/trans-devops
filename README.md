#Trans 
`/trænz/`: 是一站式开源DevOps平台，涵盖需求集成、测试集成、构建集成、策略发布等功能，有效助力产品，研发，测试和运维团队充分利用开源工具，高度可扩展的研发运维一体化管理，加速高质量软件的持续集成和持续交付。
